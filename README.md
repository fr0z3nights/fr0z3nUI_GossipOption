# fr0z3nUI GossipOption

Configurable auto-selection of gossip options, with a small UI.

## Install
1. Copy the folder `fr0z3nUI_GossipOption` into:
	- `World of Warcraft/_retail_/Interface/AddOns/`
2. Launch WoW and enable the addon.

## Slash Commands
- `/fgo` — open/toggle window
- `/fgo <id>` — open window and prefill a gossip option id
- `/fgo list` — print current gossip options (npc + option IDs)

## SavedVariables
- Account: `AutoGossip_Acc`, `AutoGossip_Settings`, `AutoGossip_UI`
- Character: `AutoGossip_Char`, `AutoGossip_CharSettings`

## Notes
- Designed to avoid unsafe interactions while in combat.
